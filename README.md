# Maze Shooter Demo

A compact third-person shooter prototype featuring **adaptive mazes**, **enemy AI behaviors**, and a **complete gameplay loop** — developed as a personal learning project in **Unreal Engine 5**, integrating built-in systems with custom maze and AI logic.


---

## Features

- Adaptive maze with trigger-based wall movement
- Enemy Behavior Tree: investigate, chase, return
- Shooting and health system 
- Basic locomotion animation system using Animation Blueprint
- Objective: reach the exit and respawn logic
- Main Menu & Finish Menu
- Windows standalone build  


---
